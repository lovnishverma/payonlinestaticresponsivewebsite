Security Issues in Online Transactions

Measures to Ensure Security

  • Encryption — Protects transmitted data by converting it into a secure code.
  • Digital Signature — Confirms authenticity through encryption and authentication.
  • Security Certificates — Verifies website or user identity using digital IDs.

Security Protocols in Internet Transactions

Secure Socket Layer (SSL)

SSL ensures safe data transmission by providing:

Authentication
Encryption
Integrity
Non-repudiation

Secure Hypertext Transfer Protocol (SHTTP)

SHTTP enhances HTTP security with encryption, authentication, and digital signatures.

Secure Electronic Transaction (SET)

Developed by MasterCard and Visa, SET secures financial transactions through digital certificates.
