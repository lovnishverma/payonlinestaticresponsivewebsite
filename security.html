Security Issues in Online Transactions

Measures to Ensure Security

  • Encryption — Safeguards data transmitted over the network by encrypting it with a secret code.
  • Digital Signature — Ensures the authenticity of information using encryption and password authentication.
  • Security Certificates — Verifies the identity of websites or users through unique digital IDs.

Security Protocols in Internet

Secure Socket Layer (SSL)

SSL is the most commonly used protocol that ensures:

  • Authentication
  • Encryption
  • Integrity
  • Non-reputability

Secure Hypertext Transfer Protocol (SHTTP)

SHTTP extends HTTP with public key encryption, authentication, and digital signatures, ensuring security for end-users.

Secure Electronic Transaction (SET)

Developed by MasterCard and Visa, SET is used by financial institutions to issue digital certificates and ensure secure electronic commerce.
